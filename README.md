## ΏθΟχ How It Works

1. **Home Screen** έΑΦ User selects Light/Dark theme and clicks Start
2. **Quiz Screen** έΑΦ 10 questions load one at a time; 15-second timer starts automatically
3. **Answering** έΑΦ Clicking an option locks in the answer, highlights correct/wrong, enables Next button
4. **Timer runs out** έΑΦ Correct answer is highlighted automatically, user can proceed
5. **Result Screen** έΑΦ Final score shown with animated ring, breakdown of correct/wrong, percentage
